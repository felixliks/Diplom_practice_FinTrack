# Дипломный проект: Веб-приложение для управления личными финансами «FinTrack»

## Актуальность темы
В современном мире управление личными финансами становится важным навыком. 
Многие люди сталкиваются с проблемой неконтролируемых расходов и отсутствием 
накоплений. Существующие приложения либо сложны в использовании, либо имеют 
платную подписку. Разработка простого и бесплатного веб-приложения позволит 
пользователям легко отслеживать доходы и расходы, что способствует финансовой 
грамотности населения.

## Целевая аудитория
- Студенты и молодые люди, начинающие вести бюджет
- Люди со средним доходом, желающие контролировать траты
- Семьи, планирующие совместный бюджет

## Предполагаемые функциональные возможности
1. Регистрация и авторизация пользователей
2. Добавление доходов и расходов с указанием категории
3. Просмотр баланса и истории операций
4. Фильтрация операций по дате и категории
5. Визуализация расходов в виде диаграммы (круговая или столбчатая)

## Используемые технологии 
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Хранение данных**: LocalStorage / IndexedDB (или Firebase для облачного хранения)
- **Визуализация**: Chart.js
- **Контроль версий**: Git, GitHub

## Ожидаемый результат
Функционирующий прототип веб-приложения с интерфейсом для учета личных финансов, 
позволяющий пользователю добавлять транзакции, просматривать баланс и анализировать 
расходы с помощью диаграммы. Приложение будет иметь адаптивный дизайн и работать 
во всех современных браузерах.
